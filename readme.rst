
============
 Weight App
============


install
=======
virtualenv env_weight
. env_weight/bin/activate
pip install -r pip-requirements.txt


run (testing)
=============
. env_weight/bin/activate
cd weight
python main.py


run tests
=========
. env_weight/bin/activate
cd weight/tests
python maintest.py
