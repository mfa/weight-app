This app was written out of a personal need. I want to track my weight every day using my Android smartphone. Because of this need the Add weight is optimized for touch input.

The sourcecode of the whole application will be available on http://github.com/mfa/weight-app/

The app is written in Python using

  • Flask
  • Jinja2
  • Flask-SQLAlchemy, sqlalchemy
  • Flask-WTF, wtforms
  • Flask-Script
  • Flask-Login
  • Flask-Testing
  • lxml
and of course bootstrap.
